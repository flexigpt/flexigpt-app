# ADR Template

## Title

`ADR: {decision title}`

## Status

`Proposed`

## Context

- Problem or change being addressed
- `[observed]` facts and constraints
- `[inferred]` implications
- Scope and complexity
- Assumptions or missing inputs, if any

## Decision

- Chosen architecture
- Scope now vs later
- Boundaries, ownership, and contracts
- Eventing decision, if any

## Rejected options

- `{option}`: why not now
- Reconsideration trigger

## Trade-offs

- Benefits gained
- Costs accepted
- Complexity intentionally avoided

## Consequences

- Operational guardrails
- Risks and mitigations
- Migration or rollback path

## Revisit triggers

- Concrete threshold or condition that reopens the decision
