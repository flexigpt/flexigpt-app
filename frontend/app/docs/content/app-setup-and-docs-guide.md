## Configure a provider

1. Open **Settings** from the sidebar.
2. Under **Auth Keys**, add at least one provider API key.
3. Save the key and return to **Chats**.

FlexiGPT does not bill you directly. Model usage is billed by the provider account connected to your API key.

## Start your first chat

1. Open **Chats** and create or select a conversation.
2. Pick a provider and a **Model Preset** from the composer controls.
3. Write your prompt.
4. Optionally attach files, images, PDFs, directories, or URLs.
5. Send the message and review the streamed response.

## Main areas of the app

- **Chats** is the main workspace for conversations, attachments, and saved history.
- **Settings** is where you manage auth keys, theme, and debug settings.
- **Model Presets** store model defaults such as temperature, token limits, reasoning settings, and system prompts.
- **Prompt Bundles** store reusable prompt templates with variables.
- **Assistant Presets**, **Skills**, and **Tools** help you manage reusable capabilities available in the app.

## Attachments and conversation history

- Text and code files can be attached as in-context text.
- Images are sent as binary blobs.
- PDFs can be extracted as text or sent as files depending on the selected mode.
- URLs and directories can also be attached to a conversation.
- Conversations are stored locally and can be reopened later.

## Using the built-in docs

- Open **Docs** from the sidebar any time.
- Use the docs section list to jump to topics quickly.
- These markdown docs are bundled into the app at build time, so they ship with the desktop app and do not depend on an external docs server.
